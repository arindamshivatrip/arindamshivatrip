## Selected work

### SnaPT  
A Chrome extension for making long ChatGPT conversations lighter, faster, and easier to navigate.  
Built around payload trimming, cached history recovery, grouped conversation turns, and a calmer control surface.

**Focus:** browser engineering, interaction design, performance, privacy-aware product decisions  
**Stack:** JavaScript, Chrome Extension APIs

---

### Personal Portfolio  
A portfolio and case-study system for presenting work across AI systems, UX research, interaction design, and XR.  
Designed as a structured proof-of-work layer rather than a static résumé page.

**Focus:** frontend architecture, content systems, visual hierarchy, responsive interaction  
**Stack:** Next.js / React, TypeScript, Tailwind CSS

---

### Media Framing & Sentiment Transformation Pipeline  
A research-oriented pipeline for converting unstructured media text into structured sentiment and framing representations.

**Focus:** NLP pipelines, LLM evaluation, bias awareness, structured outputs  
**Stack:** Python, NLP, LLM-based analysis

---

### Music & Lyrics Transcription System  
An AI-driven system for extracting lyrical and melodic content from uploaded audio, with reliability evaluation across different audio conditions.

**Focus:** audio ML, ASR, model evaluation, frontend clarity  
**Stack:** Python, React, SpeechBrain, Spleeter

---

### ShooT IT  
A real-time AR laser tag system combining physical hardware, multiplayer gameplay, haptics, and Unity-based interaction logic.

**Focus:** AR systems, multiplayer sync, physical computing, interaction feedback  
**Stack:** Unity3D, C#, MQTT, Vuforia

---

### TrailTales  
A local discovery app designed to surface non-touristy experiences through map-based exploration and recommendation-driven browsing.

**Focus:** mobile product design, map interaction, local discovery  
**Stack:** Flutter, Firebase, Google Maps API, Figma
